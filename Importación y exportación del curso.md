# Importación y exportación del curso
En este apartado se genera una copia de seguridad del curso con el objetivo de poder exportarlo, guardarlo o restaurarlo en otro momento.
Para realizar la exportación, se accede a:

Administración → Cursos → Copias

Desde esta sección se selecciona el curso correspondiente y se inicia el proceso de creación de la copia de seguridad, que incluye los contenidos, actividades y configuraciones del curso.
Esta operación permite disponer de un archivo de respaldo que puede utilizarse para restaurar el curso en el mismo sitio o en otra instalación de Moodle.
