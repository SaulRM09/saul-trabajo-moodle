# Creación de questionarios
En este apartado se crea un cuestionario utilizando preguntas previamente organizadas en el banco de preguntas, y posteriormente se verifica su funcionamiento desde los roles de estudiante y profesor.

## Organización del banco de preguntas
En primer lugar, se organizan las preguntas en categorías diferentes dentro del banco de preguntas.
Esta estructura permite mantener un orden claro y facilita la selección de preguntas al crear el cuestionario.

## Creación del cuestionario
Una vez organizadas las categorías, se crea un cuestionario y se añaden preguntas directamente desde el banco.
De este modo, el cuestionario queda configurado con preguntas clasificadas y listas para ser respondidas.

# Verificación
Para comprobar que el cuestionario funciona correctamente:

1. Se accede con un usuario estudiante y se responden todas las preguntas.
2. A continuación, se entra con el usuario profesor para revisar las respuestas y verificar las calificaciones generadas por Moodle.

Este proceso permite confirmar que el cuestionario está bien configurado, que las preguntas se califican correctamente y que los roles funcionan según lo previsto.
