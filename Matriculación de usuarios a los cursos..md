# Matriculación de usuarios a los cursos
En este apartado se configuran los métodos de inscripción de los cursos A y B, ajustando su accesibilidad y los roles de los usuarios según las necesidades del proyecto.

### Curso A
Para este curso se desactivan todos los métodos de inscripción con el objetivo de hacerlo público.
De esta forma, el curso puede ser accedido sin necesidad de iniciar sesión, permitiendo que cualquier usuario lo visualice libremente.

### Curso B
En este curso se activa el método de registro manual de usuarios, lo que permite controlar quién puede acceder al contenido.
Una vez habilitado este método, se procede a:

1. Matricular al usuario Bob como profesor del curso.
2. Matricular a los alumnos restantes como estudiantes.
