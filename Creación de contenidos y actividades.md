# Creación de contenidos y actividades
## Curso A
En este apartado se configuran los contenidos y actividades del *Curso A*, asignando los roles correspondientes y añadiendo diferentes recursos para completar la estructura del curso.

## Asignación de roles y matrícula
En primer lugar, se asigna un profesor al curso y se matricula a los alumnos que deben participar en él.
Esta configuración permite gestionar correctamente el acceso y las funciones de cada usuario dentro del curso.

## Añadir contenidos y actividades
A continuación, se incorporan diferentes tipos de actividades y recursos para construir el contenido del curso.
Entre los elementos añadidos se incluyen:

1. Actividades variadas disponibles en Moodle.
2. Recursos informativos para complementar el aprendizaje.
3. Una tarea con fecha de entrega abierta que solicita la carga de un archivo PDF por parte del alumno.

Estas acciones permiten estructurar el Curso A con materiales prácticos y actividades que facilitan el seguimiento y evaluación del alumno.

## Curso B
En este apartado se clona el contenido del Curso A dentro del Curso B, con el objetivo de reutilizar la misma estructura y actividades sin necesidad de crearlas de nuevo.

Para realizar esta acción, se accede a:

Administración del curso → Importar

Desde esta sección se selecciona el Curso A como origen y se importan todos sus contenidos al Curso B.

Esta operación permite replicar la organización, los recursos y las actividades del curso original, facilitando la creación de cursos similares sin repetir el trabajo manual.
