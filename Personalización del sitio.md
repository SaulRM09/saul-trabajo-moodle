# Personalización del sitio
### Cambio de aspecto
En este apartado se realiza el cambio de aspecto del sitio Moodle mediante la instalación y activación de un tema nuevo. Además, se personalizan diferentes elementos visuales del sitio para adaptar su apariencia a las necesidades del proyecto.

Para este proyecto se ha utilizado el tema por defecto de Moodle, al que se le han añadido elementos personalizados, como una imagen de fondo, entre otros ajustes visuales.

Estas modificaciones permiten mejorar la presentación general del sitio y darle un aspecto más coherente con los objetivos del proyecto.

### Instalación y activación del tema

Para instalar un nuevo tema, se accede a:

Administración del sitio → Plugins → Instalar complemento

Una vez instalado, el tema se selecciona desde:

Administración del sitio → Apariencia → Selector de temas

### Personalización del sitio
Después de activar el tema, se procede a modificar los elementos principales de la interfaz:

Cabecera (header)

Pie de página (footer)

Página principal (Front page)
