# Creación y gestión de usuarios
### Creación manual
En este apartado se realiza la creación manual de un nuevo usuario dentro del sitio Moodle. El objetivo es añadir un usuario llamado Bob, utilizando el método de autenticación manual, para poder asignarlo posteriormente a cursos o realizar pruebas dentro de la plataforma.

Para crear este usuario, debemos acceder a:

Administración del sitio → Usuarios → Cuentas → Añadir un usuario

Desde esta sección se pueden definir todos los datos del nuevo usuario, como el nombre, el correo electrónico, la contraseña y el método de autenticación.

# Creación masiva de alumnos
En este apartado se realiza la creación masiva de estudiantes utilizando un archivo CSV.
Este método permite añadir varios usuarios de forma rápida y eficiente, ideal para cursos con muchos participantes.

Para generar los alumnos, debemos acceder a:

Administración del sitio → Usuarios → Cargar usuarios

En esta sección se puede subir un archivo CSV con los datos de los estudiantes.
Es recomendable consultar el ejemplo de archivo CSV disponible en la propia sección de Usuarios para asegurarse de que el formato es correcto.

Una vez cargados los 10 alumnos, se procede a eliminar dos de ellos utilizando la opción:

Acciones con usuarios en bloque

Esta herramienta permite seleccionar varios usuarios y aplicar acciones masivas, como la eliminación.
