# Creació de cursos

En este apartado se lleva a cabo la creación de los cursos principales del sitio Moodle.
Se han configurado dos cursos distintos, cada uno con una estructura de temas diferente, para organizar el contenido de forma clara y progresiva.

Los cursos creados son:

Curso A, compuesto por 3 temas.

Curso B, compuesto por 5 temas.

Para crear estos cursos, debemos acceder a:

Administración del sitio → Gestiona cursos y categorías  
o bien desde el menú de navegación:

Cursos → Añadir curso

Desde esta sección es posible definir el nombre del curso, el número de temas, la descripción y otros parámetros esenciales para su funcionamiento.
