# saul-trabajo-moodle
---

## Introducción

Este proyecto consiste en instalar Moodle en una máquina Linux y configurarlo como si fuéramos los administradores de la plataforma. Durante el proceso realizamos tareas de gestión como la creación de cursos, usuarios, contraseñas seguras, categorías, cuestionarios e insignias, entre otras funciones.
El objetivo es aprender a administrar un entorno Moodle real y comprender cómo se organiza y gestiona un aula online.
